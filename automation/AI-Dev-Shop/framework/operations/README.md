# Operations

Operational reminders and implementation conventions for day-to-day pipeline execution.

Key files:

- `reminders.md` — startup prompts and one-time operator setup reminders
- `pipeline-quickstart.md` — startup, invocation, and human checkpoint reference
- `plain-language-explanations.md` — how the framework should explain its current step and internal terms to users
- `interaction-modes.md` — Agent Direct consensus and cross-agent consultation rules
- `routing-guards.md` — command-level routing guards for debate, cowork, and similar workflows
