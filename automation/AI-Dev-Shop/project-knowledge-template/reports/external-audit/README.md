# External Audit Reports

Use this retained folder only when the user explicitly wants to keep an external-audit artifact as reusable project evidence.

Default local-only paths:

```text
project-knowledge-template/.local-artifacts/external-audit/
  packets/
  runs/
  offloads/
```

Retained layout:

```text
project-knowledge-template/reports/external-audit/
  packets/
  runs/
```

Packets summarize the work handed to external auditors. Run reports capture:

- what work was audited
- what each external auditor said
- where auditors agreed, disagreed, or caught different issues
- what the Coordinator agreed with
- what should change
- what the Coordinator disagreed with and why
