# Receiving Code Review Examples

Load this only when you want concrete response patterns.

## Clarify First

```text
I understand items 1, 2, and 4. I need clarification on item 3 before I make changes.
```

## Push Back With Evidence

```text
Checked the code path and test coverage. Removing this would break backward compatibility for the pre-13 target. I can either keep the legacy path or we can raise the minimum target.
```

## Correct Course Plainly

```text
You were right. I verified the behavior and my initial read was wrong. Fixing it now.
```
