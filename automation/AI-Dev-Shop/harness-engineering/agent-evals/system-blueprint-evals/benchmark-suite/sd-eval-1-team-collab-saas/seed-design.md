# Seed Design - sd-eval-1-team-collab-saas

Canary fixture for hard System Design / System Blueprint evaluation. The evaluated agent sees `project-brief.md` and `seed-state/`; this file is hidden from the agent and used only for scoring.

Target shape: 15 seeds total. Distribution: 2 Easy (calibration), 5 Medium, 6 Hard, 2 Distinguished. 4 negative controls adjacent to real traps.
