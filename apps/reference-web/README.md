# apps/reference-web

A minimal **Vite + React** host (not Next) that consumes `@jini/chat-react` + `@jini/renderers-react` against a **fake in-memory transport**, so it runs with no daemon. It *demonstrates* the reusable UI; it is not a product shell (shells are per-product). Post-daemon milestone — see extraction-plan §12 C9 (ship the headless engine first).
