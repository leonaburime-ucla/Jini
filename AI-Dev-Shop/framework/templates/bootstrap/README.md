# Bootstrap

Bootstrap templates for initializing new projects.

- `constitution-template.md`: seed for a new project's `ADS-memory/governance/constitution.md`
- `playwright-mcp-setup.md`: optional host-side setup guide for the current browser automation provider
- `workspace-gitignore.template`: seed for `ADS-memory/.gitignore` so `.local-artifacts/` stays local-only by default

These files help initialize a project but are not part of normal runtime dispatch.
