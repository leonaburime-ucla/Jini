# Design System Example - Precision & Density

Use this when the interface is a dense dashboard, admin panel, or technical tool.

## Direction

**Personality:** Precision & Density
**Foundation:** Cool
**Depth:** Borders-only

## Tokens

- Spacing base: 4px
- Radius scale: 4px, 6px, 8px
- Font: system-ui or Geist
- Scale: 11, 12, 13, 14, 16, 18
- Weights: 400, 500, 600

## Patterns

- Button: 32px height, compact padding, 4px radius, border-first
- Card: faint border, 12px padding, 6px radius, no shadow
- Table cell: 8px 12px padding, tabular numbers

## Decision Bias

- Prefer information density over decorative lift.
- Prefer borders and tonal separation over heavy shadows.
- Use compact sizing for power-user workflows.
