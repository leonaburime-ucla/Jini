# Design System Example - Warmth & Approachability

Use this when the interface should feel friendly, collaborative, or consumer-oriented.

## Direction

**Personality:** Warmth & Approachability
**Foundation:** Warm
**Depth:** Subtle shadows

## Tokens

- Spacing base: 4px
- Radius scale: 8px, 12px, 16px
- Font: Inter or another approachable UI face
- Scale: 13, 14, 15, 16, 18, 20, 24
- Weights: 400, 500, 600

## Patterns

- Button: 40px height, comfortable padding, 8px radius
- Card: soft shadow, 20px padding, rounded corners
- Input: 44px height, generous padding, visible focus state

## Decision Bias

- Prefer soft depth and generous whitespace.
- Prefer readable, inviting surfaces over technical density.
- Use this when the product benefits from calm and friendliness.
