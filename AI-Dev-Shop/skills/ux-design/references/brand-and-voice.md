# Brand and Voice

Use this reference when the design work includes identity direction, messaging behavior, or visual tone.

## When to Use

- greenfield product with no established brand system
- redesign where the user wants a visual reset
- public-facing marketing or landing surfaces
- copy or tone decisions that materially affect UX

## Brand Foundation Inputs

If enough context exists, define:

- purpose
- vision
- mission
- values
- personality traits
- audience
- competitive differentiation

Keep this lightweight unless the user explicitly requests full brand strategy work.

## Voice Rules

Define:

- tone adjectives
- preferred reading level
- helpfulness vs confidence balance
- what copy should avoid
- how errors, empty states, and success messages should sound

Prefer clear, human, concise language over generic corporate filler.

## Visual Identity Translation

Translate brand direction into:

- palette behavior
- typography mood
- imagery treatment
- icon/illustration style
- density and whitespace
- motion character

## Messaging Architecture

For public-facing surfaces, define:

- headline style
- subhead style
- value proposition structure
- CTA tone
- proof/trust elements

## Anti-Patterns

- inventing deep brand strategy when only feature-level UI is needed
- robotic generic copy
- brand tone that conflicts with task clarity
- visual identity choices that reduce readability or accessibility

## Output Checklist

- brand assumptions stated
- tone/voice rules stated
- copy style examples provided when relevant
- visual identity translation documented
- conflicts with existing brand noted explicitly
